模型原理

基本定义

我们默认向量均为列向量，用小写字符表示向量，大写字符表示矩阵.

假设训练集中总共有 mmm 个样本，第 iii 个样本对应的特征和标签记为 (x(i),y(i))(\boldsymbol{x}^{(i)}, y^{(i)})(x(i),y(i))，不指定具体样本，可简记为 (x,y)(\boldsymbol{x}, y)(x,y)，对于每个样本的特征和标签有以下相关定义：

  • 在不同公式中 x\boldsymbol{x}x 的维数稍有不同，假设每个特征具有 nnn 个属性值，每个属性值分别为 x1,x2,⋯ ,xnx_1,x_2,\cdots,x_nx1​,x2​,⋯,xn​：
    1. x∈Rn\boldsymbol{x}\in \R^{n}x∈Rn，标准的特征向量，x=(x1,x2,⋯ ,xn)T\boldsymbol{x} = (x_1,x_2,\cdots,x_n)^Tx=(x1​,x2​,⋯,xn​)T.
    2. x∈Rn+1\boldsymbol{x}\in \R^{n+1}x∈Rn+1，增加偏置项的特征向量，x=(1,x1,⋯ ,xn)T\boldsymbol{x} = (1,x_1,\cdots, x_n)^Tx=(1,x1​,⋯,xn​)T，为了简化公式所用.
      上述中两种记法会在公式下方进行说明，若不加说明，则默认使用第一种记法.
  • 在不同问题中标签的性质有所不同：
    1. 回归问题中 y∈Ry\in \Ry∈R；
    2. 二分类问题中 y∈{0,1}y\in \{0, 1\}y∈{0,1}；
    3. 多分类问题中 y∈{0,1}K\boldsymbol{y}\in \{0, 1\}^Ky∈{0,1}K，其中 KKK 表示分类类别的总数目，y\boldsymbol{y}y 是对应类别的one-hot向量.

one-hot向量：以K=5K=5K=5为例，类别为 333，则该类别对应的one-hot向量为 [0 0 0 1 0]T[0\ 0\ 0\ 1\ 0]^T[0 0 0 1 0]T（类别从 000 开始编号）.

以下符号名称具有特别的含义：

  • y^\hat{y}y^​ 表示模型预测的结果.
  • θ\thetaθ 表示模型中包含的参数.
  • L(y;x,θ)\mathcal{L}(y;\boldsymbol{x},\theta)L(y;x,θ) 表示损失函数，简写为 L(θ)\mathcal{L}(\theta)L(θ)，书上记为 c(θ)c(\theta)c(θ).
  • J(y;x,θ)\mathcal{J}(y;\boldsymbol{x}, \theta)J(y;x,θ) 表示成本函数，也称风险函数，简记为 J(θ)\mathcal{J}(\theta)J(θ).
  • 下文中函数对向量或者矩阵求偏导，使用的均为分母布局，即求导结果的维数和求导对象的维数相同.
    例如，θ=(θ1,θ2,⋯ ,θn)T∈Rn\boldsymbol{\theta} = (\theta_1,\theta_2,\cdots,\theta_n)^T\in\R^nθ=(θ1​,θ2​,⋯,θn​)T∈Rn，函数J(θ):Rn→R\mathcal{J}(\boldsymbol{\theta}):\R^n\to \RJ(θ):Rn→R，则

    ∂J(θ)∂θ=(∂J∂θ1,∂J∂θ2,⋯ ,∂J∂θn)T\frac{\partial \mathcal{J}(\boldsymbol{\theta})}{\partial \boldsymbol{\theta}} = \left(\frac{\partial \mathcal{J}}{\partial \theta_1}, \frac{\partial \mathcal{J}}{\partial \theta_2}, \cdots, \frac{\partial \mathcal{J}}{\partial \theta_n} \right)^T ∂θ∂J(θ)​=(∂θ1​∂J​,∂θ2​∂J​,⋯,∂θn​∂J​)T

  • 上式还可记为 ∇θJ(θ)\nabla_{\boldsymbol{\theta}}\mathcal{J}(\boldsymbol{\theta})∇θ​J(θ)，表示 J(θ)\mathcal{J}(\boldsymbol{\theta})J(θ) 对向量 θ\boldsymbol{\theta}θ 求梯度.

关于更多向量或矩阵导数内容，可以参考The matrix cookbook中Derivatives篇章.

1.线性回归

标签 y∈R\boldsymbol{y}\in \Ry∈R 为数值常量，令参数向量 θ=(θ0,θ1,⋯ ,θn)T\boldsymbol{\theta} = (\theta_0,\theta_1,\cdots,\theta_n)^Tθ=(θ0​,θ1​,⋯,θn​)T，则线性模型预测值为

y^=θ0+θ1x1+θ2x2+⋯+θnxn=θTx\hat{y} = \theta_0+\theta_1x_1+\theta_2x_2+\cdots+\theta_nx_n = \boldsymbol{\theta}^T\boldsymbol{x} y^​=θ0​+θ1​x1​+θ2​x2​+⋯+θn​xn​=θTx

此处的 x\boldsymbol{x}x 为增加偏置项的特征向量.

MSE损失函数

线性回归中常用均方误差(Mean square error, MSE)作为损失函数，MSE与其对应的成本函数定义如下：

L(θ)=(θTx−y)2,J(θ)=MSE(θ)=1m∑i=1m(θTx−y)2\mathcal{L}(\boldsymbol{\theta}) = (\boldsymbol{\theta}^T\boldsymbol{x} - y)^2,\qquad \mathcal{J}(\boldsymbol{\theta}) = MSE(\boldsymbol{\theta})= \frac{1}{m}\sum_{i=1}^m(\boldsymbol{\theta}^T\boldsymbol{x} - y)^2 L(θ)=(θTx−y)2,J(θ)=MSE(θ)=m1​i=1∑m​(θTx−y)2

通过标准方程求闭解

我们将全部的特征和标签分别表示为特征矩阵和标签向量的形式：

X=[1x1(1)x2(1)⋯xn(1)1x1(2)x2(2)⋯xn(2)⋮⋱⋮⋮1x1(m)x2(m)⋯xn(m)],y=[y(1)y(2)⋮y(m)]\boldsymbol{X} = \left[\begin{matrix} 1&x^{(1)}_1&x^{(1)}_2&\cdots&x^{(1)}_n\\ 1&x^{(2)}_1&x^{(2)}_2&\cdots&x^{(2)}_n\\ \vdots&\ddots&\vdots&\vdots\\ 1&x^{(m)}_1&x^{(m)}_2&\cdots&x^{(m)}_n\\ \end{matrix}\right],\qquad \boldsymbol{y} = \left[\begin{matrix} y^{(1)}\\ y^{(2)}\\ \vdots\\ y^{(m)} \end{matrix}\right] X=⎣⎢⎢⎢⎢⎢⎡​11⋮1​x1(1)​x1(2)​⋱x1(m)​​x2(1)​x2(2)​⋮x2(m)​​⋯⋯⋮⋯​xn(1)​xn(2)​xn(m)​​⎦⎥⎥⎥⎥⎥⎤​,y=⎣⎢⎢⎢⎢⎡​y(1)y(2)⋮y(m)​⎦⎥⎥⎥⎥⎤​

于是MSE成本函数又可以表示为以下形式：

MSE(θ)=1m(Xθ−y)T(Xθ−y)MSE(\theta) = \frac{1}{m}(\boldsymbol{X}\boldsymbol{\theta}-\boldsymbol{y})^{T}(\boldsymbol{X}\boldsymbol{\theta}-\boldsymbol{y}) MSE(θ)=m1​(Xθ−y)T(Xθ−y)

对 θ\boldsymbol{\theta}θ 求导可以得到：

∂MSE(θ)∂θ=2mXT(Xθ−y)\frac{\partial MSE(\boldsymbol{\theta})}{\partial \boldsymbol{\theta}} = \frac{2}{m}\boldsymbol{X}^T(\boldsymbol{X}\boldsymbol{\theta}-\boldsymbol{y}) ∂θ∂MSE(θ)​=m2​XT(Xθ−y)

令导数等于 0\boldsymbol{0}0 可得

θ^=(XTX)−1XTy(1.1)\hat{\boldsymbol{\theta}} = (\boldsymbol{X}^T\boldsymbol{X})^{-1}\boldsymbol{X}^T\boldsymbol{y}\tag{1.1} θ^=(XTX)−1XTy(1.1)

由于 ∂MSE(θ)∂θ∂θT=XXT\frac{\partial MSE(\boldsymbol{\theta})}{\partial \boldsymbol{\theta}\partial \boldsymbol{\theta}^T} = \boldsymbol{X}\boldsymbol{X}^T∂θ∂θT∂MSE(θ)​=XXT 矩阵范数恒 ⩾0\geqslant 0⩾0（这里的矩阵范数可以是任意一种），于是 MSE(θ)MSE(\boldsymbol{\theta})MSE(θ) 是关于 θ\boldsymbol{\theta}θ 的凸函数，所以 (1.1)(1.1)(1.1) 式得到的 θ^\hat{\boldsymbol{\theta}}θ^ 就是使成本函数最小的 θ\boldsymbol{\theta}θ 值.

注意：XTX\boldsymbol{X}^T\boldsymbol{X}XTX 不一定满秩，所以可以不存在逆矩阵，但是可以通过SVD分解得到伪逆 X+X^+X+（Moore-Penrose逆矩阵）代替 (XTX)−1XT(\boldsymbol{X}^T\boldsymbol{X})^{-1}\boldsymbol{X}^T(XTX)−1XT.

求解逆矩阵的时间复杂度一般为 O(n2.4)\mathcal{O}(n^{2.4})O(n2.4) 到 O(n3)\mathcal{O}(n^3)O(n3)，而Scikit-Learn中的SVD分解复杂度约为 O(n2)\mathcal{O}(n^2)O(n2). 不适合用于处理特征数目较大的情形.

梯度下降

成本函数对 θ\boldsymbol{\theta}θ 的每一维的偏导数为：

∂MSE(θ)∂θj=2m∑i=1m(y^(i)−y(i))xj(i)(1.2)\frac{\partial MSE(\boldsymbol{\theta})}{\partial \theta_j} = \frac{2}{m}\sum_{i=1}^m(\hat{y}^{(i)}-y^{(i)})x_{j}^{(i)}\tag{1.2} ∂θj​∂MSE(θ)​=m2​i=1∑m​(y^​(i)−y(i))xj(i)​(1.2)

其中 y^(i)=θTx(i)\hat{y}^{(i)} = \boldsymbol{\theta}^T\boldsymbol{x}^{(i)}y^​(i)=θTx(i). 记为梯度向量的形式可以如下表示：

∇θMSE(θ)=2mXT(Xθ−y)\nabla_{\boldsymbol{\theta}}MSE(\boldsymbol{\theta}) = \frac{2}{m}\boldsymbol{X}^T(\boldsymbol{X}\boldsymbol{\theta}-\boldsymbol{y}) ∇θ​MSE(θ)=m2​XT(Xθ−y)

如果我们以 ∇θMSE(θ)\nabla_{\boldsymbol{\theta}}MSE(\boldsymbol{\theta})∇θ​MSE(θ) 作为每次的下降方向，设学习率为 η\etaη，则参数 θ\boldsymbol{\theta}θ 的更新方法为如下形式：

θ(下一步)=θ−η∇θMSE(θ)\boldsymbol{\theta}^{(\text{下一步})} = \boldsymbol{\theta} - \eta\nabla_{\boldsymbol{\theta}}MSE(\boldsymbol{\theta}) θ(下一步)=θ−η∇θ​MSE(θ)

梯度下降法分为三种，批量梯度下降，随机梯度下降，小批量梯度下降，三者的区别仅在每次计算梯度使用的训练集大小上：

  • 批量梯度下降(BGD)：每次以整个训练集计算一次梯度，对参数进行更新.
  • 随机梯度下降(SGD)：每次从训练集中随机选取一个样本计算梯度，对参数进行更新.
  • 小批量梯度下降(Mini-BGD)：设定一个 batch_size 大小，以 batch_size 大小对以打乱的训练集划分为多个 batch，每次取一个 batch 中的样本计算一次梯度，对参数进行更新.（好处：可以利用GPU并行运算加快更新速度）

注意：使用梯度下降法必须对样本特征的属性值大小比例近似，为了避免数值较大的属性值产生“狭长的山谷”，从而减缓训练速度. 所以需要进行归一化或者标准化处理（特征缩放）.

2.多项式回归

注意：这不是多项式插值.

首先考虑特征的属性值只有一个的情况，在线性插值中，我们的模型预测值为 y^=θ0+θ1x\hat{y} = \theta_0 + \theta_1xy^​=θ0​+θ1​x，而在 ddd 次的多项式回归中，预测模型转化为

y^=θ0+θ1x+θ2x2+⋯θdxd\hat{y}= \theta_0+\theta_1x+\theta_2x^2+\cdots\theta_dx^d y^​=θ0​+θ1​x+θ2​x2+⋯θd​xd

相当于在原有属性 xxx 的基础上扩充进新的属性值 x2, x3, ⋯ , xdx^2,\, x^3,\, \cdots,\, x^dx2,x3,⋯,xd.

当属性值个数为 n=2n=2n=2 的时候，d=3d=3d=3 次多项式回归，需要考虑不同属性之间的组合，所以全部属性值为 x13, x12x2, x1x22, x23, x12, x1x2, x22, x1, x2, 1x_1^3,\, x_1^2x_2,\, x_1x_2^2,\, x_2^3,\ x_1^2,\, x_1x_2,\, x_2^2,\, x_1,\, x_2,\, 1x13​,x12​x2​,x1​x22​,x23​, x12​,x1​x2​,x22​,x1​,x2​,1，总共 10=(2+32)10 = \binom{2+3}{2}10=(22+3​) 个.

类似地，当 n=3, d=2n=3,\ d=2n=3, d=2 时，全部属性值为 x12,x1x2,x1x3,x22,x2x3,x32,x1,x2,x3,1x_1^2, x_1x_2, x_1x_3, x_2^2, x_2x_3, x_3^2, x_1, x_2, x_3, 1x12​,x1​x2​,x1​x3​,x22​,x2​x3​,x32​,x1​,x2​,x3​,1，总共 10=(3+23)10 = \binom{3+2}{3}10=(33+2​) 个.

可以证明，当特征属性个数为 nnn，最高幂次为 ddd 的多项式回归中，最终扩充得到的属性值有 (n+dn)=(n+d)!n!d!\binom{n+d}{n}=\frac{(n+d)!}{n!d!}(nn+d​)=n!d!(n+d)!​ 个.

求解多项式扩充得到的属性值个数

首先考虑 nnn 个属性值时，齐次为 mmm 次的组合总共有多少，我们可以将问题转化为小球染色问题：总共有 mmm 个球，染上 nnn 中颜色，可以有颜色一次都不用.

该问题，就是可以有空位的隔板法，总共有 n−1n-1n−1 个隔板，隔出来 nnn 个空间，每个空间中的小球个数表示顺次表示每个属性值的幂次，由于可以有空位，所以先补上 nnn 个小球，然后转化为没有空位的隔板法，所以总共有 (m+n−1n−1)\binom{m+n-1}{n-1}(n−1m+n−1​) 中组合.

然后对于 ddd 次多项式回归，包含从 0∼d0\sim d0∼d 次的全部齐次项，所以总共有

∑m=0d(m+n−1n−1)=(d+nn)=(n+d)!n!d!\sum_{m=0}^d\binom{m+n-1}{n-1} = \binom{d+n}{n} = \frac{(n+d)!}{n!d!} m=0∑d​(n−1m+n−1​)=(nd+n​)=n!d!(n+d)!​

上述第一个等号使用了 曲棍球恒等式(Hockey Stick Identity)，证明思路：每次固定选取一个元素，然后在选取集合中选取剩余的元素，并依次递增选取集合.

弹性网络

弹性网络的成本函数为

其中 rrr 称为混合比.

  • 当 r=0r=0r=0 时，弹性网络等价于岭回归.
  • 当 r=1r=1r=1 时，弹性网络等价于Lasso回归.

总结

正则化有总比没有强，避免使用纯线性回归. 岭回归作为默认选择，当实际用到的特征数目较少，可以倾向于Lasso回归或者弹性网络，因为它们会将无用的特征权重降为 000.

一般来说，弹性网络优于Lasso回归，因为当特征数量超过训练集大小，或者几个特征强相关时（梯度范数一直较大），Lasso回归可能非常不稳定（发散）.

4.Softmax回归

我们对logistic回归进行推广到多维分类问题上，设总共有 KKK 个类别.

标准方程求闭解

构造出带有偏置项的特征矩阵 XXX 后，通过闭解式得到最优的参数向量：θ^=(XTX)−1XTy\hat{\boldsymbol{\theta}} = (\boldsymbol{X}^T\boldsymbol{X})^{-1}\boldsymbol{X}^T\boldsymbol{y}θ^=(XTX)−1XTy.

  • np.ones((n, m)) 可以创建形状为 (n, m) 全为1的矩阵.
  • np.c_[a, b] 可以将矩阵 a,b 按照列进行并排放置.
  • np.linalg.inv() 可以求解逆矩阵.

或者使用 sklearn 中的 sklearn.linear_model.LinearRegression 使用SVD分解方法求解 XTX\boldsymbol{X}^T\boldsymbol{X}XTX 的伪逆.

通过调用 intercept_ 获得截距（偏置项），coef_ 获得系数.

而 sklearn.linear_model.LinearRegression 是基于 np.linalg.lstsq() 最小二乘函数得到的，也可通过 np.linalg.pinv() 直接计算伪逆.

2.多项式回归

使用 sklearn.preprocessing.PolynomialFeatures(degree, include_bias=False) 可以生成最大幂次为 degree 的全部特征组合，include_bias=False 表示不包含偏置项，在线性回归的时候默认带了偏置，此处就无须该项.

使用 sklearn.preprocessing.StandardScaler 进行标准化处理，sklearn.pipeline.Pipeline 生成数据预处理和模型训练的流水线.

正则化线性模型

岭回归

使用 sklearn.linear_model.Ridge(alpha) 可以实现岭回归，使用 l2l_2l2​ 正则项，其中 alpha 是正则化系数.

Lasso回归

使用 sklearn.linear_model.Lasso 实现Lasso回归，使用 l1l_1l1​ 正则项，用法与 Ridge 类似.

弹性网络

使用 sklearn.linear_model.ElasticNet(l1_ratio) 实现弹性网络，其中 l1_ratio 表示 l1l_1l1​ 范数所占的权重，正则化项的具体表示形式如下：

α(l1ratio∣∣w∣∣1+12(1−l1ratio)∣∣w∣∣22)\alpha \left(l_{1_{ratio}} ||w||_1 + \frac{1}{2} (1 - l_{1_{ratio}})||w||^2_2\right) α(l1ratio​​∣∣w∣∣1​+21​(1−l1ratio​​)∣∣w∣∣22​)

3.Logistic回归

使用经典Iris数据集，包含150个鸢尾花，来自三个不同品种（山鸢尾，变色鸢尾，弗吉尼亚鸢尾），数据特征属性总共4个，分别为包含萼片的长度与宽度，花瓣的长度和宽度.

先使用花边宽度单个属性值对品种是否是弗吉尼亚鸢尾花进行预测，再使用花瓣宽度和长度两个属性值进行预测，分别绘制出决策边界和样本点图像.

使用 sklearn.linear_model.LogisticRegression(C) 实现Logistic回归，默认使用 l2l_2l2​ 正则化项，正则化系数为 1/C1/C1/C，如果不想使用正则化，可将 CCC 设置为较大值.

4.Softmax回归

也是使用 sklearn.linear_model.LogisticRegression(multi_class='multinomial')，只需将 multi_class 设置为 multinomial 即可使用Softmax回归求解多分类任务（否则可能使用“一对多(OvR)”分类策略）

特征属性仍然选取两个属性：花瓣长度，花瓣宽度. 分类目标为三种鸢尾花，绘制预测为“变色鸢尾”的等概率边界线.
